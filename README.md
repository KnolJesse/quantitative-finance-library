# Quantitative Finance Library

A modern C++20 quantitative finance library implementing pricing engines,
Monte Carlo simulation, stochastic models, and risk analytics.

## Current Features

### Pricing Engines
- Black-Scholes analytical pricer
- Monte Carlo pricer

### Products
- European call options
- European put options

### Risk Measures
- Delta
- Gamma
- Vega
- Theta
- Rho

### Models
- Geometric Brownian Motion

## Build

Requires:
- C++20 compiler
- CMake
- GoogleTest
